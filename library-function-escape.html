Configuration

Configure the following to use collection functions.

1. Add this artifact in maven dependency

  1.  <dependencies>
  2.    <dependency>
  3.      <groupId>org.jspringbot</groupId>
  4.      <artifactId>jspringbot-expression</artifactId>
  5.    </dependency>
  6.  </dependencies>

2. Import spring-expression in jspringbot-global.xml

  1. <import resource="classpath:spring/spring-expression.xml"/>
