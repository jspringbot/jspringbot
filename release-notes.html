Release Notes

Release Version 1.5.3 NEW

  • October 02, 2016

Support for saucelabs.

Release Version 1.5.1

  • September 25, 2016

Added support for file upload on http. We have added the following keywords.

  • Create HTTP File Upload Request creating new http post with file upload support.
  • Add HTTP Request File Upload add a file upload form element.
  • Create HTTP File Upload Request creating new http post with file upload support.
  • Add HTTP Request File Upload add a file upload form element.
  • Invoke HTTP Request Expect Error invokes HTTP request and expect error.
  • WaitTillElementDisappear waits until element specified with locator disappears on current page.
  • Amend Date amends current date stored.
  • currentDayOfMonth() returns the current date of month.
  • currentMonth() returns the current month.
  • currentYear() returns the current year.
  • currentTimeMillis() returns the current time in millis.
  • Parse URL parses URL.
  • Assert Protocol asserts URL Protocol.
  • Assert Host asserts URL Host.
  • Assert Port asserts URL Port.
  • Assert Path asserts URL Path.
  • Assert Paramater asserts parameter key value pair exist.
  • Assert Paramater Count asserts parameter count.
  • Assert Paramater Key Exist asserts parameter key exist.

Release Version 1.5

  • September 24, 2016

Minor release with the following changes.

  • Upgraded to Robotframework 3.0
  • Upgraded JDK to 8.
  • Upgraded Selenium to 2.53.1.
  • Support for new jspringbot maven plugin configuration splitLogs for splitting of logs.
